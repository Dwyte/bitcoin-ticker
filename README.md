# bitcoin-ticker
A system tray app for linux that shows bitcoin's price live.  
## Setup
You'll need git, pip, and some dependencies installed.  
`pip install pygobject`  

**Clone the repo**  
`git clone https://github.com/Dwyte/bitcoin-ticker.git`  
**Go the Directory**  
`cd bitcoin-ticker`  
**Run**  
`python coin.py`
